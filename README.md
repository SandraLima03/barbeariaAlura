# Projeto Barbearia Alura

## Barbearia Alura é um projeto construído através do curso básico de HTML e CSS da Alura.

### Descrição 

Nesse projeto foi construído um site completo de uma barbearia, com três páginas e três links externos para acessar cada página (páginas: home, produtos e contatos). Site criado apenas com elementos do HTML e estilizados pelo CSS.
